# Awesome Agent Plugin [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

> A curated list of awesome plugins, tools, markets, and MCP servers for AI Agents.

## 🤖 What is an Agent Plugin?
Plugins give AI Agents "hands and feet"—allowing them to browse the web, write code, control apps, and interact with the real world.

## 🏛️ Official Marketplaces
*Where major frameworks host their tool ecosystems.*

- [LangChain Community Tools](https://github.com/langchain-ai/langchain/tree/master/libs/community/langchain_community/tools) - Rich tool ecosystem for LangChain.
- [LlamaIndex Tools](https://llamahub.ai/) - LlamaHub data loaders and agent tools.
- [MCP Servers Registry](https://github.com/modelcontextprotocol/servers) - Anthropic's official Model Context Protocol server list.

## 🌐 Web & Search
*Plugins that connect agents to the live internet.*

- [Tavily AI](https://tavily.com/) - Search engine optimized for LLMs and Agents.
- [Jina Reader](https://jina.ai/reader/) - Converts any URL into clean markdown for LLMs.

## 💻 Developer Tools
*Plugins for code execution, Git, and DB operations.*

- [E2B Sandbox](https://github.com/e2b-dev/E2B) - Secure cloud sandboxes for AI agents to run code.
- [GitHub MCP Server](https://github.com/modelcontextprotocol/servers/tree/main/src/github) - Give agents the ability to review code and manage PRs.
